# Typing Speed Test

A modern, interactive typing speed test application built with HTML, JavaScript, and Tailwind CSS. This application helps users improve their typing speed and accuracy through real-time feedback and performance metrics.

## Features

- Real-time typing speed measurement (WPM - Words Per Minute)
- Live accuracy tracking
- Visual feedback for correct and incorrect keystrokes
- Clean and modern UI with Tailwind CSS
- Responsive design that works on all devices
- Instant performance metrics
- Reset functionality to try again

## Technologies Used

- HTML5
- JavaScript (ES6+)
- Tailwind CSS
- Modern CSS features (Flexbox, Grid)

## How to Use

1. Open `index.html` in your web browser
2. Start typing the displayed text in the input area
3. Your WPM and accuracy will be calculated in real-time
4. After completing the text, you can click "Try Again" to start over

## Performance Metrics

- **WPM (Words Per Minute)**: Calculated based on the number of words typed and time taken
- **Accuracy**: Percentage of correctly typed characters
